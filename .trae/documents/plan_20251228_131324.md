# 加班申请状态显示修复方案

## 需求分析
用户希望加班申请列表中的状态显示使用不同的文字颜色：
- 待审批：红色字体
- 已批准：绿色字体  
- 已拒绝：黄色字体

## 实现方案

1. **修改状态显示样式**：
   - 移除当前的徽章背景色类（bg-*）
   - 使用文字颜色类替代（text-danger、text-success、text-warning）
   - 保持简洁的显示方式，确保文字清晰可见

2. **更新模板代码**：
   - 修改 `templates/attendance/overtime_application_list.html` 文件中的状态显示部分
   - 根据状态值应用不同的文字颜色类
   - 保留状态文字的中文显示

3. **具体实现**：
   ```html
   <td>
       <span class="text-{{ 
           'danger' if application.status == 'pending' else 
           'success' if application.status == 'approved' else 
           'warning' if application.status == 'rejected' else 'secondary' 
       }}">
           {{ 
               '待审批' if application.status == 'pending' else 
               '已批准' if application.status == 'approved' else 
               '已拒绝' if application.status == 'rejected' else 
               (application.status or '未知状态') 
           }}
       </span>
   </td>
   ```

## 技术要点
- 使用 Bootstrap 5 的文字颜色类
- 根据状态值动态切换颜色
- 保持代码的可读性和可维护性
- 确保在各种状态下都有合理的显示

## 预期效果
- 待审批状态显示为红色文字
- 已批准状态显示为绿色文字
- 已拒绝状态显示为黄色文字
- 其他状态显示为灰色文字
- 文字清晰可见，符合用户要求
