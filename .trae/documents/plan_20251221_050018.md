# 每日工作安排模块开发计划

## 技术栈
- 后端：Python + Flask
- 数据库：SQLite
- 前端：HTML + CSS + JavaScript + Bootstrap
- 数据可视化：Chart.js（用于统计分析）

## 数据模型设计

### 1. 工作任务模型（WorkTask）
- id：主键
- title：任务标题
- description：任务描述
- assigned_to：分配人员
- assigned_type：分配类型（employee/contractor）
- status：任务状态（pending/in_progress/completed）
- created_at：创建时间
- updated_at：更新时间
- completed_at：完成时间

### 2. 工作日报模型（WorkReport）
- id：主键
- task_id：外键，关联工作任务
- report_date：报告日期
- content：日报内容
- author：报告作者
- created_at：创建时间
- updated_at：更新时间

## 路由设计

### 1. 工作任务管理
- GET /work-task/：工作任务列表页
- GET /work-task/add/：添加工作任务页面
- POST /work-task/add/：提交添加工作任务
- GET /work-task/edit/<int:id>/：编辑工作任务页面
- POST /work-task/edit/<int:id>/：提交编辑工作任务
- POST /work-task/delete/<int:id>/：删除工作任务
- POST /work-task/status/<int:id>/：更新任务状态

### 2. 工作日报管理
- GET /work-report/：工作日报列表页
- GET /work-report/add/<int:task_id>/：添加工作日报页面
- POST /work-report/add/<int:task_id>/：提交添加工作日报
- GET /work-report/edit/<int:id>/：编辑工作日报页面
- POST /work-report/edit/<int:id>/：提交编辑工作日报
- POST /work-report/delete/<int:id>/：删除工作日报

### 3. 统计分析
- GET /work-statistics/：工作统计分析页面

## 功能实现

### 1. 发布每日工作任务
- ✅ 创建工作任务表单
- ✅ 支持填写任务标题、描述
- ✅ 支持选择任务状态

### 2. 任务分配
- ✅ 支持从员工列表中选择班组人员
- ✅ 支持手动填写承包商信息
- ✅ 通过assigned_type字段区分分配类型

### 3. 任务状态跟踪
- ✅ 支持三种状态：未开始、进行中、已完成
- ✅ 支持通过按钮快速更新任务状态
- ✅ 显示任务状态变更时间

### 4. 工作日报填写和提交
- ✅ 支持为每个任务填写工作日报
- ✅ 支持查看任务的所有日报
- ✅ 支持编辑和删除日报

### 5. 工作统计和分析
- ✅ 显示任务状态统计（饼图）
- ✅ 显示任务完成趋势（折线图）
- ✅ 显示人员工作量统计（柱状图）

## 模板设计

### 1. 工作任务模板
- 任务列表页：显示所有任务，支持按状态筛选
- 添加/编辑任务页：任务表单，支持分配人员选择
- 任务详情页：显示任务详情和关联的工作日报

### 2. 工作日报模板
- 日报列表页：显示所有日报，支持按日期和任务筛选
- 添加/编辑日报页：日报表单

### 3. 统计分析模板
- 统计分析页：显示各种统计图表
