# 日常预防维护模块开发计划

## 技术栈
- 后端：Python + Flask
- 数据库：SQLite
- 前端：HTML + CSS + JavaScript + Bootstrap
- 数据可视化：Chart.js（用于统计分析）
- 导出功能：使用pandas库实现Excel导出

## 数据模型设计

### 1. 设备模型（Equipment）
- id：主键
- name：设备名称
- model：设备型号
- serial_number：设备序列号
- location：安装位置
- description：设备描述
- created_at：创建时间
- updated_at：更新时间

### 2. 维护计划模型（MaintenancePlan）
- id：主键
- equipment_id：外键，关联设备
- plan_name：计划名称
- description：计划描述
- maintenance_type：维护类型
- frequency：维护频率（daily/weekly/monthly/quarterly/yearly）
- next_maintenance_date：下次维护日期
- created_by：创建人
- created_at：创建时间
- updated_at：更新时间

### 3. 维护记录模型（MaintenanceRecord）
- id：主键
- plan_id：外键，关联维护计划
- equipment_id：外键，关联设备
- maintenance_date：维护日期
- executor：执行人
- content：维护内容
- result：维护结果（success/failure）
- notes：备注
- created_at：创建时间
- updated_at：更新时间

## 路由设计

### 1. 设备管理
- GET /maintenance/equipment/：设备列表页
- GET /maintenance/equipment/add/：添加设备页面
- POST /maintenance/equipment/add/：提交添加设备
- GET /maintenance/equipment/edit/<int:id>/：编辑设备页面
- POST /maintenance/equipment/edit/<int:id>/：提交编辑设备
- POST /maintenance/equipment/delete/<int:id>/：删除设备

### 2. 维护计划管理
- GET /maintenance/plan/：维护计划列表页
- GET /maintenance/plan/add/：添加维护计划页面
- POST /maintenance/plan/add/：提交添加维护计划
- GET /maintenance/plan/edit/<int:id>/：编辑维护计划页面
- POST /maintenance/plan/edit/<int:id>/：提交编辑维护计划
- POST /maintenance/plan/delete/<int:id>/：删除维护计划
- POST /maintenance/plan/export/：导出维护计划
- POST /maintenance/plan/import/：导入维护计划

### 3. 维护记录管理
- GET /maintenance/record/：维护记录列表页
- GET /maintenance/record/add/<int:plan_id>/：添加维护记录页面
- POST /maintenance/record/add/<int:plan_id>/：提交添加维护记录
- GET /maintenance/record/edit/<int:id>/：编辑维护记录页面
- POST /maintenance/record/edit/<int:id>/：提交编辑维护记录
- POST /maintenance/record/delete/<int:id>/：删除维护记录

### 4. 统计分析
- GET /maintenance/statistics/：维护统计分析页面

### 5. 提醒功能
- GET /maintenance/reminders/：维护提醒列表页

## 功能实现

### 1. 制定设备预防维护计划
- ✅ 支持创建和管理维护计划
- ✅ 支持设置维护频率和下次维护日期
- ✅ 支持关联到具体设备

### 2. 维护任务的自动提醒
- ✅ 显示即将到期的维护任务
- ✅ 支持按日期筛选提醒
- ✅ 显示维护任务的详细信息

### 3. 记录维护执行情况
- ✅ 支持为每个维护计划添加维护记录
- ✅ 记录维护日期、执行人、内容、结果等信息
- ✅ 支持查看历史维护记录

### 4. 维护记录查询和统计
- ✅ 支持按设备、日期、维护类型等条件查询
- ✅ 显示维护记录统计图表
- ✅ 显示维护计划执行情况

### 5. 维护计划的导入导出
- ✅ 支持导出维护计划为Excel文件
- ✅ 支持从Excel文件导入维护计划
- ✅ 支持导入导出设备信息
