# 经验分享模块开发计划

## 技术栈
- 后端：Python + Flask
- 数据库：SQLite
- 前端：HTML + CSS + JavaScript + Bootstrap
- 多媒体处理：Flask-Uploads
- 搜索功能：SQLAlchemy ORM查询

## 数据模型设计

### 1. 文章分类模型（ArticleCategory）
- id：主键
- name：分类名称
- description：分类描述
- created_at：创建时间
- updated_at：更新时间

### 2. 文章模型（Article）
- id：主键
- title：文章标题
- content：文章内容
- category_id：外键，关联文章分类
- author：作者
- views：浏览次数
- likes：点赞数
- created_at：创建时间
- updated_at：更新时间

### 3. 文章多媒体模型（ArticleMedia）
- id：主键
- article_id：外键，关联文章
- file_path：文件路径
- file_type：文件类型（图片/视频）
- created_at：创建时间

### 4. 评论模型（Comment）
- id：主键
- article_id：外键，关联文章
- content：评论内容
- author：评论作者
- created_at：创建时间

### 5. 点赞模型（Like）
- id：主键
- article_id：外键，关联文章
- user：点赞用户
- created_at：创建时间

## 路由设计

### 1. 文章管理
- GET /experience/：经验分享首页，显示文章列表
- GET /experience/category/<int:category_id>/：按分类显示文章
- GET /experience/article/<int:id>/：文章详情页
- GET /experience/add/：添加文章页面
- POST /experience/add/：提交添加文章
- GET /experience/edit/<int:id>/：编辑文章页面
- POST /experience/edit/<int:id>/：提交编辑文章
- POST /experience/delete/<int:id>/：删除文章

### 2. 评论管理
- POST /experience/article/<int:id>/comment/：添加评论
- POST /experience/comment/delete/<int:id>/：删除评论

### 3. 点赞管理
- POST /experience/article/<int:id>/like/：点赞/取消点赞

### 4. 搜索功能
- GET /experience/search/：搜索文章

## 功能实现

### 1. 文章发布
- 支持富文本编辑器（使用CKEditor或TinyMCE）
- 支持图片上传
- 支持视频上传

### 2. 文章分类
- 预设分类：维修技巧、故障案例、技术分享
- 支持动态添加分类

### 3. 文章查询
- 按分类查询
- 按关键词搜索

### 4. 评论功能
- 支持匿名评论
- 显示评论时间
- 支持删除评论

### 5. 点赞功能
- 支持点赞/取消点赞
- 显示点赞数

### 6. 多媒体支持
- 支持图片上传和显示
- 支持视频上传和播放
- 限制文件大小和格式

## 前端页面设计

### 1. 经验分享首页
- 显示文章列表，支持分页
- 显示分类导航
- 搜索框

### 2. 文章详情页
- 显示文章内容
- 显示多媒体内容
- 评论区
- 点赞按钮

### 3. 添加/编辑文章页面
- 富文本编辑器
- 分类选择
- 多媒体上传
