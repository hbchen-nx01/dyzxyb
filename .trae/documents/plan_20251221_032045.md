# 班组管理系统（Python版）实现计划

## 技术栈选择
- 后端：Python + Flask（轻量级web框架，适合快速开发）
- 数据库：SQLite（轻量级，无需额外安装）
- 前端：HTML + CSS + JavaScript + Bootstrap（简单易用，响应式设计）
- 图片存储：本地文件系统

## 项目结构
```
班组管理系统/
├── app.py                 # 主应用文件
├── config.py              # 配置文件
├── models.py              # 数据模型
├── routes/                # 路由模块
│   ├── __init__.py
│   └── honor_board.py     # 光荣榜路由
├── templates/             # HTML模板
│   ├── base.html          # 基础模板
│   └── honor_board/       # 光荣榜相关模板
│       ├── index.html     # 光荣榜首页
│       ├── add.html       # 添加条目
│       └── edit.html      # 编辑条目
├── static/                # 静态文件
│   ├── css/               # CSS样式
│   ├── js/                # JavaScript脚本
│   └── uploads/           # 上传的图片
└── requirements.txt       # 依赖列表
```

## 功能实现

### 1. 数据模型设计
- **HonorBoard模型**：id, 人员姓名, 奖励类型, 奖励内容, 获得时间, 图片路径, 创建时间, 更新时间

### 2. 核心功能
#### 显示功能
- 光荣榜首页显示所有奖励和表彰信息
- 支持按时间、人员、奖励类型查询

#### 管理功能
- 添加光荣榜条目
- 编辑光荣榜条目
- 删除光荣榜条目

#### 图片功能
- 支持图片上传和展示
- 支持点击图片放大查看
- 支持打开图片文件夹

### 3. 路由设计
- GET /honor-board/：光荣榜首页，显示所有条目，支持查询
- GET /honor-board/add：添加条目页面
- POST /honor-board/add：提交添加条目
- GET /honor-board/edit/<id>：编辑条目页面
- POST /honor-board/edit/<id>：提交编辑条目
- POST /honor-board/delete/<id>：删除条目
- GET /uploads/<filename>：访问上传的图片

### 4. 前端页面
- 光荣榜首页：显示所有条目，包含查询表单
- 添加/编辑页面：表单提交，包含图片上传功能
- 图片展示：支持点击放大，支持打开图片文件夹
