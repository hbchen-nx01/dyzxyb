# 班组管理系统

## 项目简介
这是一个基于Python Flask开发的班组管理系统，目前包含光荣榜模块，支持添加、编辑、删除光荣榜条目，以及图片上传和展示功能。

## 技术栈
- 后端：Python + Flask
- 数据库：SQLite
- 前端：HTML + CSS + JavaScript + Bootstrap

## 功能模块

### 光荣榜
- 显示班组人员的奖励和表彰信息
- 支持按时间、人员、奖励类型查询
- 支持添加、编辑、删除光荣榜条目
- 支持图片上传和展示
- 支持打开图片文件夹

## 安装和运行

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 运行应用
```bash
python app.py
```

### 3. 访问应用
打开浏览器，访问以下地址：
- 首页：http://127.0.0.1:5001
- 光荣榜：http://127.0.0.1:5001/honor-board/

## 使用说明

### 添加光荣榜条目
1. 进入光荣榜页面
2. 点击"添加条目"按钮
3. 填写表单信息，包括人员姓名、奖励类型、奖励内容、获得时间
4. 可选：上传相关图片
5. 点击"保存"按钮

### 编辑光荣榜条目
1. 在光荣榜页面找到要编辑的条目
2. 点击"编辑"按钮
3. 修改表单信息
4. 可选：重新上传图片（会替换原有图片）
5. 点击"保存"按钮

### 删除光荣榜条目
1. 在光荣榜页面找到要删除的条目
2. 点击"删除"按钮
3. 在确认对话框中点击"确定"

### 查询光荣榜条目
1. 在光荣榜页面的查询条件区域
2. 填写查询条件（可多选）
3. 点击"查询"按钮
4. 如需重置查询条件，点击"重置"按钮

### 查看和管理图片
1. 光荣榜条目卡片中的图片可以点击放大查看
2. 点击"打开图片文件夹"按钮可以直接打开图片存储目录

## 扩展说明
该系统采用模块化设计，便于后续扩展其他功能模块。如需添加新模块，只需：
1. 在routes目录下创建新的路由文件
2. 在templates目录下创建对应的HTML模板
3. 在models.py中定义数据模型
4. 在app.py中注册新的蓝图

## 注意事项
1. 图片上传大小限制为16MB
2. 支持的图片格式：png、jpg、jpeg、gif
3. 请勿删除static/uploads目录，否则图片将无法正常显示
4. 建议定期备份app.db文件，以防数据丢失
